# Chrome扩展图标设计

## 设计理念
基于扩展的功能（自定义新标签页、必应搜索、网站导航），设计一个简洁、现代、易于识别的图标。

## 设计方案

### 方案一：标签页+搜索图标组合
- **主元素**：一个打开的标签页形状
- **内部元素**：一个放大镜图标（代表搜索功能）
- **配色**：
  - 主色调：蓝色系（代表浏览器、信任）
  - 辅助色：白色（清晰可见）
  - 背景：透明
- **风格**：扁平化设计，简洁清晰

### 方案二：网格+搜索图标组合
- **主元素**：一个由小方块组成的网格（代表网站导航）
- **点缀元素**：一个放大镜图标（代表搜索功能）
- **配色**：
  - 主色调：蓝色系
  - 辅助色：白色
  - 背景：透明
- **风格**：现代简约

## 图标尺寸要求
Chrome扩展需要提供以下尺寸的图标：
- 16x16 px（浏览器工具栏）
- 32x32 px（Windows任务栏）
- 48x48 px（扩展管理页面）
- 128x128 px（Chrome Web Store）

## 设计工具推荐
1. **在线工具**：
   - Canva（简单易用，有模板）
   - Figma（专业设计，支持协作）
   - IconCreator（专门用于创建图标）
   - Flaticon（提供图标素材，可自定义）

2. **本地工具**：
   - Adobe Photoshop
   - Adobe Illustrator
   - Sketch（Mac）
   - GIMP（免费开源）

## 实现建议
1. 使用SVG格式创建图标，便于缩放和修改
2. 确保图标在小尺寸下仍然清晰可见
3. 避免使用过多细节，保持简洁
4. 测试不同背景下的显示效果
5. 遵循Chrome Web Store的图标 guidelines

## 命名建议
将图标文件命名为：
- icon16.png
- icon32.png
- icon48.png
- icon128.png

并在manifest.json中引用它们：
```json
{
  "icons": {
    "16": "icon16.png",
    "32": "icon32.png",
    "48": "icon48.png",
    "128": "icon128.png"
  }
}
```
