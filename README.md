# 自定义新标签页 Chrome 扩展

一个功能丰富的 Chrome 浏览器新标签页扩展，提供了必应搜索、可编辑的网站导航栏、背景图片更换、透明度设置和数据导入导出功能。

## 🚀 功能特性

- **必应搜索框**：集成必应搜索，快速搜索内容
- **可编辑网站导航**：
  - 添加、编辑、删除网站图标
  - 拖拽排序功能
  - 自动获取网站图标
- **背景图片**：
  - 自定义背景图片 URL
  - 恢复默认背景
- **透明度设置**：
  - 调节导航区和设置按钮的透明度
  - 实时预览效果
  - 保存设置
- **数据管理**：
  - 导出数据到 JSON 文件
  - 从 JSON 文件导入数据
  - 数据包括：网站导航、背景图片、透明度设置

### 方法二：从源码安装
1. 克隆或下载本仓库
2. 运行 `npm install` 安装依赖
3. 运行 `npm run build` 构建项目
4. 打开 Chrome 浏览器，进入 `chrome://extensions/`
5. 启用开发者模式
6. 点击 "加载已解压的扩展程序"
7. 选择项目根目录下的 `dist` 文件夹

## 🎯 使用说明

### 添加网站
1. 浏览所需收藏的网页，点击右上角本扩展图标
2. 修改网站名称、URL
3. 点击 "保存"

### 编辑网站
1. 悬停在网站图标上，点击编辑图标 (✎)
2. 修改网站信息
3. 点击 "保存" 或 "删除"

### 拖拽排序
1. 点击并按住网站图标
2. 拖拽到目标位置
3. 松开鼠标，完成排序

### 更换背景
1. 点击右上角设置按钮 (⚙️)
2. 选择 "默认背景" 或输入自定义背景图片 URL
3. 点击 "应用"

### 调节透明度
1. 点击右上角设置按钮 (⚙️)
2. 拖动透明度滑块调节
3. 实时查看效果，设置会自动保存

### 导入导出数据
1. 点击右上角设置按钮 (⚙️)
2. 点击 "导出数据" 下载 JSON 文件
3. 点击 "导入数据" 选择 JSON 文件
4. 确认覆盖当前数据

## 🔧 开发说明

### 项目结构
```
newtab/
├── src/                  # 源代码
│   ├── App.vue           # 主组件
│   ├── main.js           # 入口文件
│   ├── style.css         # 样式文件
│   └── assets/           # 静态资源
├── public/               # 公共资源
│   ├── manifest.json     # 扩展配置
│   └── icon*.png         # 扩展图标
├── dist/                 # 构建输出目录
├── package.json          # 项目配置
└── vite.config.js        # Vite 配置
```

### 开发环境设置
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview

# 打包成 CRX 文件
npm run build:crx
```
